Lore Ipsum

loremipsum.py is a python module that lets developers to generate random text. Generated text can be one or more sentences, or paragraphs, and resamble the famous dummy poetry text of the printing and typesetting industry: Lorem Ipsum.

loremipsum.py was born as first step for a larger project which will let me to build my musixmatch api mock web service: MusicaeIpsum.

Most of the code of this module was taken from lorem-ipsum-generator by James Hales. James stated that his package served his purpose and he was not interested in further development, so I took it over (to serve my purpose).

Features

loremipsum.py includes some feature more than its predecessor

  • Methods for text generation now returns tuples including some statistics.
  • New generator methods to use in for-statements
  • Module-wide functions for text generation
  • Module-wide functions for statistics-less text generation.
